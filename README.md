# Vehicles EDA

Application to work on EDA process for a vehicle's dataset

## Dependencies

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install dependencies.

```bash
pip install pandas
pip install plotly_express
pip install streamlit
```
